## 0.1.0

* Initial release.
* `AdaptiveLayout` — responsive breakpoint-based layout widget with compact, medium, expanded, and large window-size classes.
* `ShimmerContainer` — Material 3-aware shimmer loading animation with customizable base/highlight colors and duration.
* `SkeletonLoader` — shimmer-animated list placeholder with custom item builder; uses theme-aware colors for dark mode support.
* `EmptyState` — configurable empty-state widget with icon, title, description, and action button.
* `ErrorState` — configurable error widget with retry button and custom retry label.
* `AsyncButton` — `FilledButton` with built-in async loading state management and optional `onError` callback for error handling.
* `FlavorValidators` — composable form validators: `required`, `email`, `minLength`, `maxLength`, `numeric`, `compose`.
* `PullToRefresh` — Material 3 pull-to-refresh wrapper with `canRefresh`, `strokeWidth`, `triggerMode`, and `notificationPredicate` parameters.
